# Geometry With Augmented Reality

**Install the apk and scan the different figures in the Photos pdf.

## Overview

**Geometry With Augmented Reality** is an educational game developed as a Final Degree Project (TFG) focused on teaching geometry concepts to children with **ACNEE (Students with Special Educational Needs)** through interactive examples and **Augmented Reality (AR)**.

The main goal of the project is to provide an accessible and engaging learning experience where children can understand geometric concepts through visual interaction, real-world examples, and immersive AR experiences.

By combining educational methodologies with emerging technologies, the game aims to make geometry easier to understand, more attractive, and adapted to the learning needs of each student.

## Features

- 📐 **Interactive geometry learning**
  - Introduction to basic geometric concepts through visual examples.
  - Learning shapes, properties, and spatial relationships in an intuitive way.

- 📱 **Augmented Reality integration**
  - Visualization of geometric objects in the real world.
  - Interactive 3D models that help students understand abstract concepts.

- 🎮 **Game-based learning**
  - Educational activities designed as playful experiences.
  - Encourages exploration and active participation.

- ♿ **Accessibility-focused design**
  - Developed considering the needs of children with special educational needs.
  - Simple interfaces and clear visual feedback.

## Objectives

The main objectives of this project are:

- Improve the understanding of geometry concepts through visual and interactive learning.
- Facilitate access to educational content for ACNEE children.
- Use Augmented Reality as a tool to support teachers and students.
- Create a motivating environment that encourages learning through experimentation.
